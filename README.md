# Palindrome Checker app
## UC5 – Stack Based Palindrome Checker
This program checks whether a given string is a palindrome using Java’s built‑in Stack class.
## How It Works
- Push each character of the string onto a stack
- Pop characters to read them in reverse
- Compare with the original string
- If all characters match → it is a palindrome

## Example
Input: noon
Output: true
