# Palindrome Checker App
## UC11 – Object-Oriented Palindrome Service

- Demonstrates palindrome validation using object-oriented design
- Palindrome logic is encapsulated inside a PalindromeService class
- Improves:

  -- Reusability
  -- Readability
  -- Separation of concerns



## How It Works
### Main Class Flow

- Create an object of PalindromeService
- Call the method checkPalindrome(input)
- Print whether the input is a palindrome

### Palindrome Logic (Inside Service Class)

- Maintain two pointers: start and end
- Compare characters while moving inward
- If any mismatch occurs → return false
- If all match → return true

## Example

- Input: racecar
- Output: true
