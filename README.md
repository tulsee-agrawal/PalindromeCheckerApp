# Palindrome Checker App
## UC8 – LinkedList Based Palindrome Checker
This program checks whether a string is a palindrome using Java’s built‑in LinkedList.
## How It Works

### 1) Add all characters into a LinkedList<Character>
### 2)Remove characters from:

removeFirst() → from the front
removeLast() → from the back


### 3)Compare both values
If they all match → the string is a palindrome

## Example
-Input: level
-Output: true
