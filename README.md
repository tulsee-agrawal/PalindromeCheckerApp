# Palindrome Checker App
## UC6 – Queue + Stack Palindrome Checker
This program checks whether a string is a palindrome by using both a Queue (FIFO) and a Stack (LIFO).
## How It Works

### 1) Insert each character into:

a Queue (front → back)
a Stack (top)


### 2) Remove characters:

queue.poll() removes from front
stack.pop() removes from top


### 3) Compare both values
If all pairs match then the string is a palindrome

## Example
Input: civic
Output: true
## Technologies

Java
Queue<Character> using LinkedList
Stack<Character>
