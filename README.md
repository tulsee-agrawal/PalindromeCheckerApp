# Palindrome Checker App
## UC10 – Normalized Palindrome Checker

- Checks whether a string is a palindrome after normalization
- Removes spaces, punctuation, and symbols
- Converts characters to lowercase
- Uses the two‑pointer technique

## How It Works
### Normalization Steps

- Keep only letters and digits
- Convert characters to lowercase
- Build a cleaned string using StringBuilder

### Palindrome Validation

- Compare characters from both ends
- Move pointers inward
- If all match → palindrome
- If mismatch → not a palindrome

## Example

- Input: A man, a plan, a canal: Panama
- Normalized: amanaplanacanalpanama
- Output: true
