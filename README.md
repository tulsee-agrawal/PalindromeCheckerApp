# Palindrome Checker App
## UC7 – Deque Based Palindrome Checker
This program checks whether a string is a palindrome using Java’s Deque (Double‑Ended Queue).
## How It Works

### 1) Insert all characters into a Deque<Character>
### 2)Remove characters from:
removeFirst() → from the front
removeLast() → from the back

### 2)Compare both values
If all match then the string is a palindrome

## Example
Input: refer
Output: true
