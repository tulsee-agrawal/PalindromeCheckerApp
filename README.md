# Palindrome Checker App
## UC9 – Recursive Palindrome Checker
1) This program checks whether a string is a palindrome using recursion.
2) Characters are compared from the start and end, moving inward with each recursive call.
## How It Works

1) Compare characters at start and end
2)If they match then recursively call for the inner substring
3) Stop when:

. start >= end → all characters matched
. characters don’t match → not a palindrome



## Example
Input: madam
Output: true
